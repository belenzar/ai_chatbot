# 🤖 AI Chatbot

Aplicación web conversacional impulsada por inteligencia artificial, desarrollada con React e integrada con la API de Claude mediante funciones serverless. Permite a los usuarios interactuar con un asistente virtual en tiempo real a través de una interfaz moderna, rápida y responsive.

## 🚀 Demo

🔗 https://ai-chatbot-claude-gamma.vercel.app/

## 📋 Descripción

Este proyecto fue desarrollado para explorar la integración de modelos de IA generativa en aplicaciones web modernas. La aplicación utiliza React para la interfaz de usuario y funciones serverless para gestionar de forma segura las solicitudes a la API de Claude.

## ✨ Características

- Chat en tiempo real con inteligencia artificial.
- Integración segura con Claude API.
- Arquitectura serverless.
- Interfaz responsive.
- Manejo de estados de carga y errores.
- Despliegue en Vercel.

## 🛠 Tecnologías

- React
- JavaScript
- HTML5
- CSS3
- Claude API
- Serverless Functions
- Vercel

## 🏗 Arquitectura

```text
Usuario
   │
   ▼
Frontend (React)
   │
   ▼
Serverless Function
   │
   ▼
Claude API
```

Las solicitudes del usuario son procesadas mediante una función serverless que actúa como intermediaria entre el frontend y la API de Claude, manteniendo las credenciales protegidas y fuera del cliente.

## ⚙️ Instalación

### Clonar repositorio

```bash
git clone https://github.com/belenzar/ai_chatbot.git
```

### Instalar dependencias

```bash
npm install
```

### Ejecutar en desarrollo

```bash
npm run dev
```
